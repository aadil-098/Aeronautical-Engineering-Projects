# Aircraft Wing Spar Fatigue Analysis

## Overview

This project evaluates the structural integrity and fatigue life of an aircraft wing spar using ANSYS Workbench under CS-25 discrete gust loading conditions.

## Objective

To determine the maximum stress, deformation, safety factor, and fatigue life of an aircraft wing spar.

## Software Used

- ANSYS Workbench
- ANSYS Mechanical

## Methodology

- 3D CAD Model
- Mesh Generation
- Static Structural Analysis
- Fatigue Analysis
- Goodman Correction

## Results

- Maximum Von Mises Stress: **50.37 MPa**
- Static Safety Factor: **6.85**
- Fatigue Life: **1×10⁸ cycles**
- Fatigue Safety Factor: **1.64**
