# Computational Feasibility Study of 3D-Printed Self-Healing Epoxy Composites

## Overview

This project investigates the computational feasibility of self-healing epoxy composites fabricated using additive manufacturing. The study combines Computational Fluid Dynamics (CFD), multiphysics simulation, and surrogate modeling to evaluate the healing behavior of epoxy composites containing microvascular networks.

---

## Objective

To analyze the flow behavior of healing agents inside embedded microchannels and evaluate the feasibility of autonomous crack repair using computational simulation.

---

## Software Used

- COMSOL Multiphysics
- ANSYS Fluent
- Python

---

## Methodology

- Developed the computational model of self-healing epoxy composites.
- Simulated laminar flow through embedded microvascular channels.
- Evaluated capillary-driven healing agent transport.
- Generated surrogate models for rapid performance prediction.
- Compared computational results with published research.

---

## Results

- Successfully simulated healing agent flow inside microchannels.
- Evaluated pressure and velocity distribution.
- Demonstrated the feasibility of computational modeling for self-healing composites.
- Identified parameters affecting healing efficiency.
